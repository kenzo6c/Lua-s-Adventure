# Lua-s-Adventure
Action RPG in the LÃ–VE library in Lua.



KEYBINDINGS : 

*moving*
z : go up
s : go down
q : go left
d : go right

*shooting*
SPACE : shoot

*items*
a : consume potion
i : inventory (not yet implemented)

*miscellanenous*
p : parameters
l : lose life
m : regain life
LSHIFT : display hitboxes
LCTRL : display item collect radius
